# 🍻 SPINSHOT

### The Ultimate Party Spinner Game

------------------------------------------------------------------------

# 1. Product Overview

**App Name:** SPINSHOT\
**Tagline:** Spin it. Do it. Or drink it.

SPINSHOT is a real-time multiplayer party drinking game for ages
18--30.\
Players join a live room, spin a dynamic task wheel, complete
challenges, and track shots in a competitive leaderboard.

The game is fast (20--30 minutes), chaotic, social, and built for real
parties.

------------------------------------------------------------------------

# 2. Core Concept

-   One player creates a room\
-   Others join via code / QR\
-   Each player has their own phone\
-   Turns rotate automatically\
-   On your turn → Spin the wheel 🎡\
-   Get a mission\
-   Complete it → Safe\
-   Fail it → Take a shot 🥃\
-   Leaderboard tracks total drinks\
-   Random duo/group challenges appear\
-   Game ends after X rounds or time limit

# 4. Game Flow (UX)

## Welcome Screen

-   Logo animation\
-   Buttons: Create Game / Join Game / How to Play

## Create Game

-   Enter name\
-   Choose language (Hebrew / English)\
-   Choose game length (20 / 30 min)\
-   Create room

## Join Game

-   Enter code\
-   See players joining in real time\
-   Host starts game

## Game Screen Layout

-   Top: Current player name + Round counter\
-   Center: Big glowing spinner\
-   Bottom: Leaderboard preview + Spin button

## After Spin Screen

Mission appears in animated card with: - ✅ Completed\
- ❌ Failed (Drink)

Leaderboard updates instantly.
