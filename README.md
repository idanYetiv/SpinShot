# SpinShot

> Spin it. Do it. Or drink it.

Real-time multiplayer party drinking game. Players join a room via code, spin a task wheel, complete challenges or drink.

## Features

- 7 mission categories: Solo, Duo, Group, Truth, Skill, Speed, Secret
- 80 bilingual missions (English + Hebrew)
- Animated spinner wheel
- Real-time leaderboard
- Chaos events (10% chance per spin)
- End-game titles (MVP, Most Drank, Most Fearless)
- RTL support for Hebrew
- Neon party theme with glow effects

## Tech Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · Zustand · React Router · Framer Motion · i18next · Supabase · Capacitor

## Getting Started

```bash
npm install
npm run dev
```
